Explicando código c

Código

Se muestra el código para explicar


#include <stdio.h>


    int main() 
    {
        int edad;
        printf("¿Cuántos años tienes? ");
        scanf("%d", &edad);


        if (edad >= 18) {
            printf("Eres mayor de edad.\n");
        } else {
            printf("Eres menor de edad.\n");
        }


    return 0;
    }

Explicación del código

1. Librebrías y función principal

#include <stdio.h>
En esta línea importa la librería estándar de entrada y salida, permite usar printf y scanf.

int main()
Es la función principal donde el sistema inicia la ejecución del programa.

{}
lo que esta en las llaves, en general todo eso es nuestro cuerpo de la función.

2. Variables y lectura de datos

int edad;
Declara una variable de tipo entero para almacenar los años que introduzca el usuario.

printf y scanf
El comando printf muestra la pregunta en la pantalla, el scanf lee el valor ingresado por el teclado, en este caso lo guarda en la variable edad.

3. Cuerpo de la función

if (edad >= 18)
Esto es una condicional, evalúa si el valor de la edad es mayor o igual a 18. Si se cumple esta condición, el programa imprimirá en la pantalla que la persona es mayor de edad.

else
Si la condición anterior no se cumple (si es menor de 18), se ejecuta lo que está dentro de este bloque e imprime que es menor de edad.

return 0;
Indica que el programa finalizó su ejecución sin ningún error.
